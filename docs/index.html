How It Works

1

You invoke /dg

On a git diff, a file, or an entire directory. The orchestrator gathers the code.

2

Gilfoyle attacks

An independent AI agent with Gilfoyle's persona reviews your code. Finds bugs, vulnerabilities, architectural flaws. Delivers them with devastating sarcasm.

3

Dinesh defends

A second independent agent defends the code. Concedes real issues, pushes back on unfair criticism, dismisses nitpicks.

4

They debate

Back and forth until convergence — no new issues, full concession, or round cap hit. The adversarial tension surfaces issues a single reviewer would miss.

5

You get the verdict

A structured summary with banter highlights, issues categorized by debate outcome, and a clean checklist of what to fix.

6

Get the comic strip

Optionally generates an HTML comic strip of the best moments — with SVG character avatars, speech bubbles, and a score card. Attach it to your PR for the team to enjoy.

What It Checks

Not a surface-level linter. Both agents review across seven deep technical domains.

!

Security

Hardcoded credentials, PII exposure, injection attacks, buffer overflows, auth gaps, OWASP Top 10. Scans dependencies for known CVEs using native audit tools with OSV.dev and NVD fallback.

2

Database

Missing indexes, N+1 queries, connection pooling, missing transactions, schema issues, queries that work at 1K rows but die at 1M.

3

Distributed Systems

Missing retries and backoff, no idempotency, no circuit breakers, race conditions, missing timeouts, ignored partial failures.

4

Performance & KISS

Premature optimization, missing obvious optimization, over-engineered abstractions, KISS violations, memory leaks, blocking in async contexts.

5

Logging & Observability

PII in logs, missing logging in critical paths, excessive debug logging, no structured logging, missing correlation IDs, swallowed exceptions.

6

Language-Specific Best Practices

Detects the language and applies its idioms. Catches ecosystem-specific anti-patterns — Java raw types, Go error ignoring, Python mutable defaults, JS any-typing.

7

Design Pattern Fluff Detection

Spots useful patterns. Calls out fluff ruthlessly. AbstractSingletonProxyFactoryBean is not engineering — it's a cry for help.
