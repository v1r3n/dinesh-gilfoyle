# Install /dg for Claude Code

Download these three files to `~/.claude/skills/dg/`:

1. Create the directory: `mkdir -p ~/.claude/skills/dg`
2. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/SKILL.md` to `~/.claude/skills/dg/SKILL.md`
3. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/gilfoyle-agent.md` to `~/.claude/skills/dg/gilfoyle-agent.md`
4. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/dinesh-agent.md` to `~/.claude/skills/dg/dinesh-agent.md`
5. Download `https://raw.githubusercontent.com/v1r3n/dinesh-gilfoyle/main/dg/comic-template.html` to `~/.claude/skills/dg/comic-template.html`

## Verify

Tell the user to start a new Claude Code session and type `/dg` to verify the skill loads.
